Use of Selectors in CSS

Checking the html file we can see the use of different variant of selectors when defining rules for CSS. The "element" selector lets use define a rule based on a given html element e.g. "p", "body" etc.

The class selector lets us define rules based on a class. Here we use ".genericname" to define the CSS rule.

Finally we have the id selector. With this we can define a rule for a given element using their id. This is achieved using "#genericid".
